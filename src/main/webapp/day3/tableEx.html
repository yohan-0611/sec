Table Tag

  • 테이블 Tag 는 크게 3개로 구성되어집니다.
  • Table > tr(row) > td(row 내부의 Cell)
    위 태그는 반드시 순서대로 중첩 되어야 하며(안그럼 깨짐), 열리면 반드시 닫아야 합니다.
  • 꼭 기억할 것...모든 Cell은 행(Row) 내에 존재해야 합니다...즉 row 태그내에 td 태그를 사용하는 만큼 cell이 생성됩니다.
  • 대표적 속성으로는 width, height, border(보더크기), cellpadding, cellspacing 등이 있으나
    일반적으로는 css 에서 정의 후 제어 합니다.
    위 속성은 table 부터 td 까지 모두 사용가능하며, table 에 정의하면 자동으로 하위에 상속 되어집니다.

    • 테이블의 논리적 구분 Tag : Sementic Tag(body 내의 논리적으로 구분한 content 영역 tag) 처럼
    테이블에도 논리적 tag 가 존재합니다.(몰라도 그만..) thead,tbody,tfoot 로 정의될 수 있습니다.
    이 Tag 가 사용되는 부분은 만약 여러분이 인쇄를 할경우, 브라우저는 이 태그를 인식해서, 태이블이 인쇄시
    가급적 페이지가 나뉘지 않도록 사용되기 위함입니다.
