Form Tag의 개론

form tag 의 기본 속성 알아보기

  • forn tag 에는 데이터를 서버에 전송할때 전송 방식을 결정할 수 있습니다.
  • 이 전송방식을 method 라는 속성으로 정의 하는데, 기본값은 get(데이터 길이제한 있음...@4,100 bytes) 입니다
  • method 는 약 7개로 값이 정의 되어있는데,get, post, put, delete, trace,??
  • get 은 일반적으로 데이터 조회(select)를 서버에 요청시에 사용됨
  • post 는 데이터 생성(insert), 갱신(update), 삭제(delete) 를 서버에 요청시에 사용됨
  • 나머지 것들은 RESTfull Service 를 강의할 때 알게됩니다.
  • name 속성 : 폼의 이름을 지정합니다. 자바스크립트나 DOM 등을 이용해서 form 데이터를 제어할때 사용됩니다.
  • action : 전송될 서버의 URL 을 명기 합니다. 이떄 대부분의 서버페이지는 서버프로그래밍이 적용된 페이지가 됩니다.
  • enctype : 폼데이터의 encoding 타입을 명기합니다... 나중에 파일 업로드시에 보게 됩니다.
  • target : 폼데이터가 처리 된 이후에 결과를 출력할 프레임이나 창을 지정할 수 있습니다. ahchor 의 target 과 같음
